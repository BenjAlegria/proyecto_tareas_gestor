# Gestor de Tareas - Evaluacion N°2

## Descripción
Aplicación web para gestionar tareas mediante un tablero Kanban. Permite agregar tareas con descripción, prioridad y fecha límite, y moverlas entre tres estados.

## Estructura del proyecto
```
proyecto_tareas_gestor/
├── index.html
├── README.md
├── assets/
│   ├── css/
│   │   ├── style.css
│   │   └── responsive.css
│   ├── js/
│   │   └── script.js
│   ├── img/
│   └── fonts/
└── docs/
    └── uso_ia.md
```

## Tecnologías
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Agregar tareas con descripción, prioridad y fecha
- Validación de formulario
- Mover tareas entre columnas: Pendientes, En Progreso, Completadas
- Eliminar tareas
- Contadores automáticos por columna
- Diseño responsive

## Cómo usar
Clonar el repositorio en tu computador.
Abrir la carpeta con Visual Studio Code.
Abrir el archivo `index.html` en el navegador o con live server.
